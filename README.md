# Welcome

Celebrity Wine Archive is an ambitious digital museum dedicated to documenting celebrity wines from all over the world.

The project combines wine collecting, storytelling, design and technology into one searchable archive.

The long-term vision is to create the world's leading online resource for celebrity wines.

---

# Vision

Celebrity Wine Archive is **not** an online wine shop.

It is **not** just another bottle database.

It is a digital museum.

Every bottle has a story.

Every celebrity has a history.

Every collection deserves to be preserved.
